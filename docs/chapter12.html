1. 不変（Immutable）オブジェクト

table 不変コレクションの作成方法

方法コード例注意点
List.of()List.of("a", "b", "c")null要素不可。変更不可
List.copyOf()List.copyOf(existingList)既存リストの防御的コピー。変更不可
Collections.unmodifiableList()Collections.unmodifiableList(list)元リストの変更はビューに反映される点に注意
Map.of() / Map.ofEntries()Map.of("key", "value")順序不定。null不可

description 不変クラスの設計パターン

自作クラスを不変にするには以下を守ります：① クラスをfinalにする、② フィールドをprivate finalにする、③ セッターを提供しない、④ ミュータブルなフィールドは防御的コピーで返す

// 不変クラスの例
public final class Money {
    private final int amount;
    private final String currency;

    public Money(int amount, String currency) {
        if (amount < 0) throw new IllegalArgumentException("金額は0以上");
        this.amount = amount;
        this.currency = Objects.requireNonNull(currency);
    }

    public int getAmount() { return amount; }
    public String getCurrency() { return currency; }

    // 演算は新しいオブジェクトを返す
    public Money add(Money other) {
        if (!this.currency.equals(other.currency))
            throw new IllegalArgumentException("通貨が異なります");
        return new Money(this.amount + other.amount, this.currency);
    }
}

// 不変コレクション
List<String> immutable = List.of("A", "B", "C");
try {
    immutable.add("D"); // UnsupportedOperationException
} catch (UnsupportedOperationException e) {
    System.out.println("変更不可: " + e.getClass().getSimpleName());
}

// List.copyOf: 既存の可変リストから不変コピーを作成
List<String> mutable = new ArrayList<>(Arrays.asList("X", "Y"));
List<String> copied = List.copyOf(mutable);
mutable.add("Z");
System.out.println("元リスト: " + mutable); // [X, Y, Z]
System.out.println("コピー:   " + copied);  // [X, Y] (影響を受けない)
terminal Output 変更不可: UnsupportedOperationException
元リスト: [X, Y, Z]
コピー: [X, Y]

2. 防御的コピー

クラスがミュータブルなオブジェクトを受け取ったり返したりする場合、呼び出し元のコードからの意図しない変更を防ぐために防御的コピーを作成します。

import java.util.Date;

public final class Period {
    private final Date start;
    private final Date end;

    // コンストラクタ: 引数をコピーして受け取る（攻撃を防ぐ）
    public Period(Date start, Date end) {
        // new Date(d.getTime()) で防御的コピーを作成
        this.start = new Date(start.getTime());
        this.end   = new Date(end.getTime());
        if (this.start.after(this.end))
            throw new IllegalArgumentException("開始日は終了日より前でなければなりません");
    }

    // ゲッター: フィールドのコピーを返す（内部状態の直接露出を防ぐ）
    public Date getStart() { return new Date(start.getTime()); }
    public Date getEnd()   { return new Date(end.getTime()); }
}

// 検証
Date s = new Date();
Date e = new Date(s.getTime() + 10000);
Period p = new Period(s, e);

// 元のDateオブジェクトを変更してもPeriodは影響を受けない
s.setTime(Long.MAX_VALUE);
System.out.println("start改ざん後: " + p.getStart()); // 元の値が保持される
terminal Output start改ざん後: Thu May 16 14:30:00 JST 2024 (元の値が保持される)

3. インジェクション対策

table SQLインジェクション対策の比較

Statement（危険）PreparedStatement（安全）
SQLの構築方法文字列連結プレースホルダ「?」使用
入力値の扱いそのままSQLに埋め込みエスケープ処理される
攻撃への耐性脆弱安全
// NG: SQLインジェクション脆弱なコード
String userInput = "Alice' OR '1'='1"; // 悪意のある入力
String dangerousSql = "SELECT * FROM users WHERE name = '" + userInput + "'";
// 実行されるSQL: SELECT * FROM users WHERE name = 'Alice' OR '1'='1'
// → 全レコードが返ってしまう！

// OK: PreparedStatementで対策
String safeSql = "SELECT * FROM users WHERE name = ?";
try (PreparedStatement pstmt = conn.prepareStatement(safeSql)) {
    pstmt.setString(1, userInput); // 自動的にエスケープされる
    ResultSet rs = pstmt.executeQuery();
    // 'Alice'' OR ''1''=''1' として安全に処理される
}
terminal Output PreparedStatementにより、悪意ある入力はエスケープされ、
意図しないSQL実行を防ぐことができる

table パスワード等の機密情報の扱い方

事項NG（危険）OK（安全）
パスワードの保持型String passwordchar[] password
理由Stringはイミュータブルでガベージコレクション前まで文字列プールに残る可能性char[]は使用後 Arrays.fill(pw, '\0') でゼロクリア可能

// NG: Stringでパスワードを保持
String badPassword = "secret123"; // GCまでメモリに残る可能性

// OK: char[]でパスワードを保持し、使用後にクリア

5. 入力値の検証

外部からの入力（ユーザー入力、ファイル読み込み、ネットワーク等）は常に信頼しないことが原則です。

// Objects.requireNonNull: nullチェックと例外スローを1行で
public void process(String value) {
    // 従来
    if (value == null) throw new NullPointerException("valueはnullにできません");
    // 簡潔に書くなら
    String v = Objects.requireNonNull(value, "valueはnullにできません");
}

// 範囲チェック
public void setAge(int age) {
    if (age < 0 || age > 150) {
        throw new IllegalArgumentException("年齢の範囲外: " + age);
    }
    this.age = age;
}

// 入力値の正規化（パストラバーサル攻撃対策）
Path requestedPath = Path.of("/uploads").resolve(userInput).normalize();
if (!requestedPath.startsWith("/uploads")) {
    throw new SecurityException("不正なパスアクセス: " + requestedPath);
}

// null安全な文字列比較
String input = null;
// NG: input.equals("admin") → NullPointerException
// OK: 定数を先に書く
boolean isAdmin = "admin".equals(input); // false (例外なし)
System.out.println("管理者: " + isAdmin);
terminal Output 管理者: false
