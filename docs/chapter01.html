1. レコードクラス (Record Classes)

table 従来のクラスとレコードクラスの比較

特徴 従来のクラス (POJO) レコードクラス
ボイラープレート ゲッター、セッター、equals等を全て記述 不要（自動生成される）
不変性 フィールドに final を付ける必要あり 強制的に不変（暗黙の private final）
継承 (extends) 自由 不可（Recordクラスを暗黙的に継承）
インタフェース実装 可 可
フィールド取得 getName() のような命名が一般的 name() のように括弧のみ

description データを保持するクラスを簡潔に定義

Java 16で正式追加された機能です。
DTO（Data Transfer Object）など、データを格納して運ぶだけのクラスを作る際、記述量を劇的に減らすことができます。
コンパイラが自動的に equals(), hashCode(), toString()、コンストラクタ、および各フィールドのアクセッサメソッドを生成してくれます。
試験対策のポイント：レコードクラス自体は暗黙的に final であり、他のクラスを継承することはできません。 

// レコードクラスの定義 (これだけでコンストラクタや各種メソッドが生成される)
public record Person(String name, int age) {
    // 必須ではないが、コンパクトコンストラクタで値の検証が可能
    public Person {
        if (age < 0) {
            throw new IllegalArgumentException("Age cannot be negative");
        }
    }
}

public class Main {
    public static void main(String[] args) {
        Person p1 = new Person("Alice", 30);
        Person p2 = new Person("Alice", 30);
        
        // アクセッサメソッドは get~ ではなく フィールド名()
        System.out.println("Name: " + p1.name() + ", Age: " + p1.age());
        
        // toString() が自動生成され、見やすい形式で出力される
        System.out.println("toString: " + p1);
        
        // equals() もフィールドの同値性で自動比較される
        System.out.println("p1.equals(p2): " + p1.equals(p2));
    }
}
terminal Output Name: Alice, Age: 30
toString: Person[name=Alice, age=30]
p1.equals(p2): true

2. シールドクラス (Sealed Classes)

table サブクラスに必須な修飾子の比較

シールドクラスを継承するサブクラスは、必ず以下のいずれかの修飾子を付けなければなりません。

修飾子 意味 継承の制限状態
final これ以上、別のクラスに継承させない 完全にクローズ
sealed さらに permits を使って継承先を制限する 制限を継続
non-sealed 継承の制限を解除する 完全にオープン（誰でも継承可）

description 継承できるクラスを厳密に制限する

クラスやインタフェースの継承を特定のクラスのみに許可する仕組みです。sealed 修飾子と permits キーワードを使用します。
無秩序なクラスの拡張を防ぐことができ、パターンマッチングと組み合わせた際に「網羅性（すべてのサブクラスのパターンをカバーしているか）」をコンパイラがチェックできるようになります。
試験対策のポイント：許可されたサブクラスは、必ず同一モジュール（名前なしモジュールの場合は同一パッケージ）内に存在しなければなりません。 

// 親クラス：Circle と Square にのみ継承を許可
public sealed class Shape permits Circle, Square {}

// 子クラス1：これ以上継承させない (final)
public final class Circle extends Shape {
    public void draw() { System.out.println("Drawing a Circle"); }
}

// 子クラス2：制限を解除し、誰でも継承可能にする (non-sealed)
public non-sealed class Square extends Shape {
    public void draw() { System.out.println("Drawing a Square"); }
}

// （参考）Squareは non-sealed なので、さらに別のクラスが継承可能
public class ColoredSquare extends Square {}

public class Main {
    public static void main(String[] args) {
        Shape circle = new Circle();
        Shape square = new Square();
        
        System.out.println("Circle class: " + circle.getClass().getSimpleName());
        System.out.println("Square class: " + square.getClass().getSimpleName());
    }
}
terminal Output Circle class: Circle
Square class: Square
