// 定数フィールド
Locale ja = Locale.JAPAN;

2. 数値・通貨フォーマット

table NumberFormatの主なファクトリメソッド

メソッド説明
NumberFormat.getInstance(locale)汎用的な数値フォーマット
NumberFormat.getCurrencyInstance(locale)通貨フォーマット（¥, $等のシンボル付き）
NumberFormat.getPercentInstance(locale)パーセントフォーマット
NumberFormat.getIntegerInstance(locale)整数フォーマット（小数点以下を丸める）
double price = 12345.678;

// getInstance: 汎用的な数値フォーマット
NumberFormat numFmt = NumberFormat.getInstance(Locale.US);
numFmt.setMinimumFractionDigits(2);
numFmt.setMaximumFractionDigits(2);
System.out.println("数値:       " + numFmt.format(price)); // 12,345.68

// getCurrencyInstance: 通貨フォーマット（¥, $等のシンボル付き）
NumberFormat usFormat = NumberFormat.getCurrencyInstance(Locale.US);
NumberFormat jpFormat = NumberFormat.getCurrencyInstance(Locale.JAPAN);
System.out.println("USD: " + usFormat.format(price)); // $12,345.68
System.out.println("JPY: " + jpFormat.format(price)); // ¥12,346

// getPercentInstance: パーセントフォーマット
NumberFormat pctFmt = NumberFormat.getPercentInstance(Locale.US);
System.out.println("パーセント: " + pctFmt.format(0.756)); // 76%

// getIntegerInstance: 整数フォーマット（小数点以下を丸める）
NumberFormat intFmt = NumberFormat.getIntegerInstance(Locale.US);
System.out.println("整数:       " + intFmt.format(price)); // 12,346

// 文字列をパース
Number parsed = usFormat.parse("$1,234.56");
System.out.println("パース結果: " + parsed.doubleValue()); // 1234.56
terminal Output 数値: 12,345.68
USD: $12,345.68
JPY: ¥12,346
パーセント: 76%
整数: 12,346
パース結果: 1234.56

3. 日付・時刻フォーマット（DateTimeFormatter）

table DateTimeFormatterの主なパターン記号

記号意味例
yyyy年（4桁）2024
MM月（2桁、先頭ゼロ）05
M月（先頭ゼロなし）5
dd日（2桁）07
HH時（24時間形式）14
hh時（12時間形式）02
mm分30
ss秒45
aAM/PMPM
E曜日（短縮）Mon
zタイムゾーン名JST
LocalDate date = LocalDate.of(2024, 5, 7);
LocalTime time = LocalTime.of(14, 30, 45);
LocalDateTime dateTime = LocalDateTime.of(date, time);

// yyyy/MM/dd HH:mm:ss
DateTimeFormatter fmt1 = DateTimeFormatter.ofPattern("yyyy年MM月dd日");
DateTimeFormatter fmt2 = DateTimeFormatter.ofPattern("yyyy/MM/dd HH:mm:ss");
DateTimeFormatter fmt3 = DateTimeFormatter.ofPattern("MM-dd-yyyy", Locale.US);

System.out.println(date.format(fmt1));     // 2024年05月07日
System.out.println(dateTime.format(fmt2)); // 2024/05/07 14:30:45
System.out.println(date.format(fmt3));     // 05-07-2024

// hh(時12時間) + a(AM/PM)
DateTimeFormatter fmt12h = DateTimeFormatter.ofPattern("hh:mm a", Locale.US);
System.out.println(time.format(fmt12h)); // 02:30 PM

// E(曜日短縮) + z(タイムゾーン名) (注: zはZonedDateTimeが必要)
ZonedDateTime zdt = ZonedDateTime.of(dateTime, ZoneId.of("Asia/Tokyo"));
DateTimeFormatter fmtE = DateTimeFormatter.ofPattern("E, yyyy/MM/dd HH:mm z", Locale.ENGLISH);
System.out.println(zdt.format(fmtE)); // Tue, 2024/05/07 14:30 JST

// パース（文字列→日時オブジェクト）
LocalDate parsed = LocalDate.parse("2024年05月07日", fmt1);
System.out.println("パース: " + parsed); // 2024-05-07

// 定義済フォーマッタ
System.out.println(dateTime.format(DateTimeFormatter.ISO_LOCAL_DATE_TIME));
// 2024-05-07T14:30:45

4. ResourceBundle（国際化対応テキスト）

description プロパティファイルの命名規則

リソースバンドルは、ベース名 + アンダースコア + 言語コード + アンダースコア + 国コード + .properties という命名規則で配置します。

ファイル名対応Locale優先順位
messages_ja_JP.properties日本語（日本）最高（最も具体的）
messages_ja.properties日本語（共通）↑
messages.propertiesデフォルト（全言語共通）最低（フォールバック）
// ファイル配置例 (src/main/resources/ または クラスパス直下)
// messages.properties:
//   greeting=Hello
//   farewell=Goodbye
//
// messages_ja.properties:
//   greeting=こんにちは
//   farewell=さようなら

// ResourceBundleの取得と使用
Locale ja = Locale.JAPAN;
ResourceBundle bundle = ResourceBundle.getBundle("messages", ja);
// まず messages_ja_JP.properties を探し、なければ messages_ja.properties、
// なければ messages.properties の順でフォールバック

System.out.println(bundle.getString("greeting")); // こんにちは
System.out.println(bundle.getString("farewell"));  // さようなら

// キーが存在しない場合 → MissingResourceException（実行時例外）
// bundle.getString("nonexistent"); // ← 例外スロー
terminal Output こんにちは
さようなら
